# 我的个人主页

一个简洁、现代化的个人主页网站，部署在 Cloudflare Pages 上。

## 特性

- 响应式设计，支持各种设备
- 现代化渐变色设计
- 平滑滚动效果
- 动画交互
- SEO 友好

## 本地预览

直接用浏览器打开 `index.html` 文件即可预览。

## 部署到 Cloudflare Pages

查看 `DEPLOYMENT_GUIDE.md` 了解详细部署步骤。

## 自定义

编辑 `index.html` 文件，修改以下内容：

- 你的名字
- 关于我的描述
- 技能标签
- 项目信息
- 联系方式链接

## 技术栈

- HTML5
- CSS3 (渐变、动画、Grid、Flexbox)
- 原生 JavaScript (无框架依赖)

## 许可

可自由使用和修改
